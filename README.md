# Roscados y Partes Industriales - Website

Página web profesional para Roscados y Partes Industriales, SA de CV.

## 📋 Estructura del Proyecto

```
roscados-website/
├── public/
│   └── index.html          # HTML principal
├── src/
│   ├── index.js            # Punto de entrada React
│   ├── index.css           # Estilos Tailwind CSS
│   └── RoscadosWebsite.jsx # Componente principal
├── package.json            # Dependencias del proyecto
├── .env                    # Variables de entorno
├── .gitignore             # Archivos a ignorar en Git
└── README.md              # Este archivo
```

## 🚀 Cómo Usar

### 1. Descarga los archivos
Descarga todos estos archivos del proyecto.

### 2. Organiza la estructura
Crea las carpetas `public/` y `src/` y coloca los archivos en sus respectivos lugares.

### 3. Sube a GitHub
- Sube toda la carpeta a tu repositorio `roscados-website`
- Los archivos deben quedar en la raíz (sin subcarpeta adicional)

### 4. Vercel Deploy Automático
Una vez en GitHub, Vercel detectará los cambios automáticamente y hará el deploy.

Todas se instalan automáticamente cuando subes a Vercel.

## 🌐 Dominio

Después del deploy, conecta tu dominio `roscadosypartes.com`:

1. Ve a tu proyecto en Vercel
2. Settings → Domains
3. Agrega: `roscadosypartes.com`
4. Sigue las instrucciones de DNS
5. Espera 24-48h para propagación
